Technical Requirements & Installation

System Requirements

To ensure the best experience, your system should meet these specifications:

  • Operating System: Windows 10/11 or macOS 12+
  • Processor: Intel i5/AMD Ryzen 5 or better
  • Memory: 8GB RAM minimum (16GB recommended)
  • Graphics: DirectX 11 compatible GPU with 4GB VRAM
  • Storage: 10GB available space
  • Internet: Stable broadband connection

Installation Guide

Follow these steps for a smooth installation process:

  1. Download the installer from our official website
  2. Run the setup file and follow the installation wizard
  3. Create your account or sign in with existing credentials
  4. Complete the initial setup and tutorial
  5. Start creating your first cake!
